## 배열

배열 요소의 타입이 고정되어 있지 않으므로, 같은 배열에 있는 배열 요소끼리의 타입이 서로 달라도 된다. 

배열 요소의 인덱스가 연속적이지 않아도 되기 때문에 특정 배열 요소가 비어있을 수 있다.

자바 스크립트에서 배열은 Array 타입의 객체로 다뤄진다.

````js
arData[0] = 10;
arData[10000] = 1;
````

> 위의 코드처럼 0번째 인덱스, 10000번째 인덱스를 가진 배열을 만들면
>
> { 0:10, 10000:1 } 같은 맵의 형태로 배열이 만들어진다.

#### 배열의 선언

- `let 배열명 = [값1, 값2, ...];`

- `let 배열명 = new Array(값1, 값2, ...);`

- `let 배열명 = Array(값1, 값2, ...);`

#### 배열의 길이

- `배열명.length`

#### 배열의 요소

- 배열 요소 추가
  - `배열명[인덱스] = 추가할 요소;`
  - `배열명.push(추가할 요소);`
- 배열 요소 삭제
  - `배열명.pop()` : 마지막에 있는 값을 제거
  - `배열명.shift()` : 가장 처음에 있는 값을 제거
  - `배열명.splice(시작인덱스, 지울요소개수, 추가할값1, 추가할값2, ..)` : 삭제 및 추가

```` js
let arrData = [10,20,30,40];

arData.push(50);	// [10,20,30,40,50]
arData.pop();	// [10,20,30,40]
arData.shift();	// [20,30,40]
arData.splice(0,1,80);	// [80,30,40]
````

#### 배열의 연결

- `배열1.concat(배열2)` 

#### 배열 요소로 문자열 만들기

- `배열명.join("구분점");` : 각 요소들을 구분점으로 연결해서 문자열로 만들어 리턴

#### 배열의 정렬

- `배열명.sort(기준함수);`

  - 오름차순 정렬

    ````js
    let arr = [1, 8, 6, 24, 13, 75, 4, 9];
    arr.sort(function(a,b){return a-b});
    console.log(arr);
    // [1, 4, 6, 8, 9, 13, 24, 75]
    ````

  - 내림차순 정렬

    ````js
    let arr = [1, 8, 6, 24, 13, 75, 4, 9];
    arr.sort(function(a,b){return b-a});
    console.log(arr);
    // [75, 24, 13, 9, 8, 6, 4, 1]
    ````

## 대화상자 (dialog)

`window.`은 생략 가능하다.

- `window.alert()`

  사용자에게 간단한 메시지를 보여주고, 그에 대한 사용자의 확인을 기다린다.

- `window.confirm()`

  사용자에게 간단한 메시지를 보여주고, 확인을 누르면 참 취소를 누르면 거짓을 리턴한다.

  ````js
  while(true){
      if(confirm("계속 진행하시겠습니까?")){
          break;
      }
  }
  ````

  > 위의 코드를 실행하고 사용자가 확인을 누르면  `true`가 리턴되어 무한루프가 종료된다.

- `window.prompt()`

  사용자에게 간단한 메시지를 보여주고, 사용자가 입력한 문자열 값을 리턴한다.

  ````js
  let name = prompt("이름을 입력하세요.");
  document.write(name + "님 환영합니다.");
  ````

  > 위의 코드를 실행하면 '이름을 입력하세요.' 라는 메시지와 함께 입력할 수 있는 공간이 나타난다. 그곳에 문자열을 입력해 name이라는 변수에 넣어 사용할 수 있다.
